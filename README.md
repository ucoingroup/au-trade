# AusTrade Hub | 澳通商汇

> Australia-ASEAN-China Top 30 Trade Commodities Portal

🌐 **Live Site**: [https://ucoingroup.github.io/au-trade/](https://ucoingroup.github.io/au-trade/)

## Features

- 🏆 **Top 30 Commodities** — Ranked by trade volume across AU-ASEAN-CN corridor
- 🔍 **Search & Filter** — Real-time search by commodity name, HS code, or tag
- 📋 **Detail Modal** — Click any card for full dimensions (trade value, tariff, companies, events)
- 🏙️ **City Picks** — Tailored recommendations for Chongqing, Shanghai, Guangdong, Beijing
- 📊 **Price Ticker** — 10 key commodity benchmark prices
- 📰 **Key Events** — Trade timeline with latest developments
- 📜 **Policy Reference** — ChAFTA, ACFTA 3.0, RCEP, AANZFTA-2, CPTPP
- 🌍 **Bilingual** — Full Chinese/English toggle
- 📱 **Responsive** — Mobile-first design

## Tech Stack

- Pure HTML5 / CSS3 / Vanilla JS
- No frameworks or build tools
- Google Fonts (Inter + Noto Sans SC)
- GitHub Pages hosted

## Update Schedule

Data refreshes every **Sunday 10:30 AM Beijing Time** via automated cron job.
